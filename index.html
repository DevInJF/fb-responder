Novo Aplicativo

O que?

O que?

Abre aspas: Você sabe para que serve um aplicativo? Não?! Então clique aqui e conheça. O app é pré-requisito para uma notificação ser enviada.

Dê um nome para seu aplicativo, adicione uma descrição para o usuário entender do que se trata e insira os links que o usuário será redirecionado após confirmar a participação no aplicativo.

Como?

Como?

Para o aplicativo funcionar, é necessário possuir uma página de Negócios (Fan Page) no Facebook e seguir os seguintes passos abaixo para validar seu aplicativo.

Acesse developers.facebook.com/apps/e procure no canto superior direito da página a opção "Adicionar um Novo Aplicativo".

Na janela que se abre, escolha Website.

Informe um nome para o aplicativo. Escolha um nome que seja sua marca ou nicho de atuação. O usuário verá o nome escolhido. Assim que o nome for digitado clique em Criar Novo App ID do Facebook.

No passo seguinte apenas escolha a categoria do Aplicativo e clique em criar ID do Aplicativo.

Pronto. Seu aplicativo está criado. Volte para a página inicial developers.facebook.com/apps/e clique no nome do aplicativo.

E agora você verá o App ID e o App Secret. Ambas informações serão utilizadas na próxima etapa. Vamos lá? Clique em "Próximo"!

Onde?

Onde?

Agora é só copiar o App ID e o App Secret abaixo.

Ok?

Ok?
